Function to show the direction with maximum information

Description

The function plots the numerical value of the angular direction of the theta1, theta2 composite having the maximum information (i.e., where the standard error of measurement is smallest) conditional upon theta1, theta2. The theta1, theta 2 grid is split up into 81 equal size intervals (9 x 9).

Usage

direction.plot(a1, a2, d, correlation = 0, filename = "NA", type = "jpeg")

Arguments

a1 A vector of the item discrimination parameters for theta1
a2 A vector of the item discrimination parameters for theta2
d A vector of the item difficulty parameters
correlation The correlation between theta 1 and theta 2. If the correlation is not equal to zero, the a1 and a2 will be transformed to maintain the orthogonal representation of the x and y axes.
filename The user specified filename given to save the plot; if the filename is provided, the function will automatically save the plot for the users by the provided filename.
type The format of files in which the user saves the plot. Options include (wmf, emf, png,jpg,jpeg,bmp,tif,tiff,ps,eps, pdf).

Details

The direction plot provides a description of the angular direction associated with the theta1, theta2 composite providing the most information for a given values of theta1, theta2. "At each selected point is the number denoting the angular direction of the theta1, theta2 composite that is being measured most precisely. The font size of the number indicates the relative amount of information in the specified direction: the larger the font, the greater the measurement precision (Ackerman, 1996, p.319)."

Author(s)

Zhan Shu, Terry Ackerman, Matthew Burke

References
