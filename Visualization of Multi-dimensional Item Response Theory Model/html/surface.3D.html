surface.3D(MIRTG)R Documentation

Function to plot the item/test surface in three-dimensional style

Description

This function creates a three dimensional plot of either the item or test surface. The plot allows for free rotation by clicking and dragging in the plot space.

Usage

surface.3D(a1, a2, d, correlation = 0)

Arguments

a1 A vector of the item discrimination parameters for theta1
a2 A vector of the item discrimination parameters for theta2
d A vector of the item difficulty parameters
correlation The correlation between theta 1 and theta 2. If the correlation is not equal to zero, the a1 and a2 will be transformed to maintain the orthogonal representation of the x and y axes.

Details

The package is built based on "rgl" pacakge, and must be loaded before use. This function plots either the item or test response surface. The x-axis represents theta1, and the y-axis represents theta2. When an item is plotted, the surface value reflects the probability of a correct response along the z-axis. The plotted surface will be made up of two colors. The color change corresponds to the shift from less than to greater than a .5 probability of a correct response to the item. When the test surface is plotted, the surface value represents the expected number correct score along the z-axis.

Author(s)

Zhan Shu, Terry Ackerman, Matthew Burke

References

Ackerman,T.(1996),Graphical Representation of Multidimensional Item Response Theory Analyses,Applied Psychological Measurement,20(4),311-329.

d<-c( -1.11,0.29,1.51,-0.82,-1.89,-0.49,1.35,0.82,-0.21,-0.04,-0.68,0.22,-0.86,-1.33,1.21)
